### Color Palette

```
Primary Actions:
  - Join Button: Gradient Purple (#8B5CF6 to #6D28D9)
  - Confirm Button: Gradient Green (#10B981 to #059669)
  - Cancel Button: Gray (#6B7280)

Status States:
  - Success: Green (#10B981)
  - Error: Red (#EF4444)
  - Warning: Amber (#F59E0B)
  - Info: Blue (#3B82F6)

Backgrounds:
  - Card Preview: Light Purple (#F3E8FF)
  - Border: Purple (#A78BFA)
  - Modal Background: White (#FFFFFF)
  - Overlay: Black with opacity (rgba(0,0,0,0.5))

Text:
  - Primary: Dark Gray (#111827)
  - Secondary: Medium Gray (#6B7280)
  - Tertiary: Light Gray (#9CA3AF)
  - Disabled: Very Light Gray (#D1D5DB)
```

### Typography

```
Headings:
  - Modal Title: Bold, 20px
  - Section Title: Semibold, 16px
  - Card Title: Semibold, 14px

Body Text:
  - Regular: Normal, 14px
  - Small: Normal, 12px
  - Code/Monospace: Monospace font, 11px

Input Fields:
  - Label: Semibold, 12px
  - Helper Text: Normal, 11px
  - Input Font: Monospace for codes, 16px
```

### Spacing

```
Padding:
  - Modal: 16px (large)
  - Card: 16px (medium)
  - Button: 12px vertical, 20px horizontal
  - Input: 8px vertical, 12px horizontal

Margins:
  - Between elements: 16px
  - Between sections: 24px
  - Button spacing: 8px

Border Radius:
  - Modal: 8px
  - Button: 6px
  - Input: 4px
  - Card: 8px
```
